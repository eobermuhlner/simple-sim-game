List of 64x64 map objects for this medieval sim game.

The list entries are in the form
- file-name: description

The file-name is the systematic name that will be used in the game, without .png

# Terrain Objects

- tree-large: Large tree canopy viewed from top-down
- tree-small: Small tree canopy viewed from top-down
- boulder-large: Large grey boulder viewed from top-down
- boulder-small: Small grey boulder viewed from top-down
- boulder-large-snow: Large grey boulder with snow on top
